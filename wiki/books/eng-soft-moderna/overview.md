# Engenharia de Software Moderna — Overview

A comprehensive software engineering textbook by Marco Tulio Valente, covering both classical and modern approaches to building software. The book explores the fundamental challenges of software as a medium — drawing on Brooks' analysis — and builds up from requirements to architecture, processes, and quality.

## Major themes

- [[wiki/concepts/functional-vs-nonfunctional-requirements]] — the two classes of requirements every system must satisfy
- Brooks' four properties of software: complexity, conformity, easy changes, invisible

## Key concepts

- Functional vs non-functional requirements
- Brooks' essential difficulties of software
